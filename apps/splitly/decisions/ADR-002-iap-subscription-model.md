# ADR-002: IAP 수익 모델 — 구독 방식 결정
- 버전: v1.0.2
- 날짜: 2026-03
- 상태: 결정됨

## 배경
초기 v1.0.0은 단순 무료 앱. 수익화를 위해 IAP 모델 검토.

## 결정
소모성 인앱결제(기능 잠금 해제) 대신 구독 모델 채택.
- Free: 광고 노출 + 저장 버튼 수동
- Pro Monthly: ₩3,900/월 — 자동 저장 + 광고 없음
- Pro Annual: ₩29,900/년

## 영향 범위
- composeApp: IAP 구독 플로우 UI
- moon-billing-kmp: 구독 상태 관리
- AdMob: Free 사용자에게만 광고 노출
